CSS 盒模型

盒模型是 CSS 布局的基石，它定义了元素在页面上的位置和大小。在 CSS 中，每个 元素都被看作是一个矩形的盒子，这个盒子由内容、内边距、边框和外边距组成。

flex局部

选择器优先级 ：内联样式（元素的 style 属性）优先级最高，其次是ID选择器，然后是类选择器、属性选择器和伪类选择器，最后是元素选择器和伪元素选择器。可通过 !important 提升特定样式的优先级，但应谨慎使用。

盒模型 ：CSS盒模型由内容区（content）、内边距（padding）、边框（border）和外边距（margin）组成。

布局 ： display 属性决定元素的显示类型，如 block （块级元素）、 inline （内联元素）、 inline-block （内联块元素）、 flex （弹性布局）和 grid （网格布局）。
